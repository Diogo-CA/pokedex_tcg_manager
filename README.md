# Catálogo Online - Pokémon TCG

## Sobre o Projeto
Este projeto consiste no desenvolvimento de um site para o gerenciamento e organização de um catálogo online de cartas de Pokémon TCG. O objetivo principal é facilitar a organização do acervo pessoal de jogadores e colecionadores, oferecendo um ambiente digital prático para o registro de cartas, montagem de baralhos e consulta às coleções oficiais. 

Projeto desenvolvido como Trabalho Web para o curso de Ciência da Computação da Fundação Educacional do Município de Assis (FEMA) na disciplina de Linguagens de Programação, do professor Almir Camolesi.

## Funcionalidades Principais
* Registro e catalogação das cartas que o usuário possui no acervo.
* Criação, edição e agrupamento de cartas em baralhos personalizados.
* Funcionalidade de favoritar cartas de interesse (lista de desejos).
* Pesquisas e filtros avançados sobre as coleções oficiais do jogo via API.
