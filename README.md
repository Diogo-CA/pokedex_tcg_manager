# Catálogo Online - Pokémon TCG

## Sobre o Projeto
Este projeto consiste no desenvolvimento de um site para o gerenciamento e organização de um catálogo online de cartas de Pokémon TCG[cite: 1]. O objetivo principal é facilitar a organização do acervo pessoal de jogadores e colecionadores, oferecendo um ambiente digital prático para o registro de cartas, montagem de baralhos e consulta às coleções oficiais[cite: 1]. 

Projeto desenvolvido como Trabalho Web para o curso de Ciência da Computação da Fundação Educacional do Município de Assis (FEMA)[cite: 1].

## Funcionalidades Principais
* Registro e catalogação das cartas que o usuário possui no acervo[cite: 1].
* Criação, edição e agrupamento de cartas em baralhos personalizados[cite: 1].
* Funcionalidade de favoritar cartas de interesse (lista de desejos)[cite: 1].
* Pesquisas e filtros avançados sobre as coleções oficiais do jogo via API[cite: 1].

## Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, TypeScript e Bootstrap[cite: 1].
* **Back-end:** Java[cite: 1].
* **Integração de Dados:** Pokémon TCG API (TCGdex)[cite: 1].
* **Gestão e Versionamento:** GitHub e Trello[cite: 1].
